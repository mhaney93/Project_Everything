How It Works

Node

Click a node

Opens the sidebar panel to view summaries, notes, and attachments for that topic.

Node

Click the dots

Expands the node to reveal its child topics, or collapses them if already open.

🔍

Search

Use the search bar (or Ctrl+S) to jump directly to any topic on the map.

Keyboard Shortcuts

Navigation

Arrow Keys Move focus between nodes
Enter Select focused node / expand dots
Escape Clear focus highlight
Ctrl+S / Cmd+S Focus search bar

Custom Nodes (requires sign-in)

Shift+Enter Add child node to focused/selected node
Tab Add sibling node
Delete Delete selected custom node
Double-click Edit custom node name inline
Enter / Escape Save / cancel inline editing

Notes & Grids

Enter Create new note below current note
Tab Indent note or grid
Backspace (at start of note) De-indent note
Backspace (top-left grid cell) De-indent grid
Escape (empty note or grid) Delete draft note or grid

Settings & Features

Node Size

Use the + and − buttons in Settings to scale all nodes uniformly. This is useful for accessibility — it scales nodes and text independently of browser zoom. Click the percentage to reset to 100%.

Fullscreen Mode

Hides the header and expands the map to fill the viewport. Toggle from Settings, or press Escape to exit.

Other Options

  • Enable animations — Smooth transitions when nodes appear or move
  • Smooth panning — Animated camera movement when navigating
  • Auto expand — Automatically expand a node when you navigate to it

Storage Guidelines

Total limit: Each account has a 10 GB combined storage limit, shared across:

  • Files — Uploaded attachments (PDFs, images, videos, etc.)
  • Notes & Nodes — Text notes, grids, custom nodes, and summaries

Allowed file types:

  • Images: JPG, PNG, GIF, WebP
  • Documents: PDF, Word (DOC, DOCX), Excel (XLS, XLSX)
  • Videos: MP4, MPEG, MOV, AVI, FLV, MKV, WebM, 3GP, OGG
  • Text: TXT

There is no per-file size limit — only the 10 GB total matters.

Your storage usage page shows a breakdown of files vs. notes so you can see where space is going.
