Keyboard Shortcuts

  • Ctrl+S / Cmd+S Focus search bar
  • Arrow Keys Navigate between nodes
  • Enter Select focused node
  • Shift+Enter Add custom child node (when signed in)
  • Tab Add custom sibling node (when signed in)
  • Enter (inside a note) Create a new note below
  • Tab / Shift+Tab (inside note or grid) Indent / De-indent
  • Backspace (start of note) De-indent note if possible
  • Backspace (top-left grid cell at start) De-indent grid if possible
  • Delete Delete selected custom node (when signed in)
  • Double-Click Edit custom node name (when signed in)
  • Enter / Escape Save / Cancel inline editing

Storage Guidelines

Total Storage Limit: Each user account has a 10GB combined storage limit. This includes:

  • Files: Uploaded attachments (PDFs, images, videos, etc.)
  • Notes & Nodes: All text notes, grids, custom nodes, and summaries you create

Your storage usage shows a breakdown of files vs. notes/nodes so you can see what's consuming space.

Allowed File Types:

  • Images: JPG, PNG, GIF, WebP
  • Documents: PDF, Word (DOC, DOCX), Excel (XLS, XLSX)
  • Videos: MP4, MPEG, MOV, AVI, FLV, MKV, WebM, 3GP, OGG
  • Text: Plain text files (TXT)

File Size: There is no limit on individual file size, but your total storage across all files and notes cannot exceed 10GB.

Note: While text notes are typically much smaller than files, creating thousands of notes with extensive content can consume significant storage space.
