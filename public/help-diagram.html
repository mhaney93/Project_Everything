Keyboard Shortcuts

  • Ctrl+S / Cmd+S Focus search bar
  • Arrow Keys Navigate between nodes
  • Enter Select focused node
  • Shift+Enter Add custom child node (when signed in)
  • Tab Add custom sibling node (when signed in)
  • Double-Click Edit custom node name (when signed in)
  • Enter / Escape Save / Cancel inline editing
